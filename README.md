## Installation

> [!CAUTION]
> Requires [pywal](https://github.com/dylanaraps/pywal) to work

## Config Details

-   OS: Arch Linux
-   DE/WM: Hyprland
-   Shell: zsh
-   Terminla emulator: Kitty
-   Text editor: nvim
-   App launcher: rofi
-   Top bar: waybar
-   Screen locker: swaylock effects
-   Wallpaper select : swww
-   Browser: Firefox
-   Colorscheme generation utility: pywal
